Git basic commands:
---------------------------

1. Initialises the git 
     git init 
2. Add the files to git(staging area)
    

3. Commit the added files
    Git commit -m"Welcome file is pushed"

4.create a repository in GitHub

5. Create a branch
    Git branch -M main

6. Adding the remote repository with git 
    Git remote add origin https://github.com/pravalika7777/basicfull-stack

7. Push the files which are committed 
    Git push -u origin main
